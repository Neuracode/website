Intermediate Scikit-Learn

What will I learn in this course

This course is meant for students who have already learned Intermediate Python and know basic machine learning concepts. By taking the course, students will learn more about machine learning and Scikit-Learn, a machine learning library for Python that includes a lot of useful machine learning algorithms and functions. Students will use this library along with a variety of different datasets to create machine learning projects.
